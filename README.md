# Simple Calculator App

A basic web-based calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

## How to Use
1. Open `index.html` in your browser.
2. Enter two numbers.
3. Select the operation button.
4. The result will be displayed immediately.
